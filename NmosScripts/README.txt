This NMOS helper starts an NMOS registry, NMOS test sender, and an NMOS controller which connects aforementioned sender to the tachyon-live NMOS receiver.

How to run NMOS helper...
1. Start a command prompt as administrator
2. Navigate to the NmosHelper folder
3. Type run_nmos_helper into the command prompt

The script waits 3 seconds in between starting the registry and sender to make sure they start up fully. 
After this script is running, start tachyon live NMOS job.
After you are finished, to close the script + its processes, go to the original window running the script and press any key
